# ANA4315 - Introduction to Bioinformatics: CA1

## Overview of Bioinformatics
Bioinformatics is an interdisciplinary field that combines biology,
computer science, mathematics, and data science to analyze and 
interpret complex biological data. It plays a critical role in 
modern biological research, enabling scientists to process large 
datasets such as DNA sequences, protein structures, gene expression 
profiles, and evolutionary relationships. Tools and techniques from 
bioinformatics are widely applied in genomics, drug discovery, 
personalized medicine, and evolutionary biology.

## Assignment Description

- Generation of structured CSV data output for downstream analysis

## How to Run
Clone the repository and run the Bash automation script:

    git clone https://github.com/usmanarammah/ANA4315-BIOINFORMATICS-GROUP-14.git
    cd ANA4315-BIOINFORMATICS-GROUP-14
    bash automate.sh

The script will automatically execute all member scripts and 
generate the CSV output file in the output/ directory.
